## 5. 积分获取与兑换流程

```mermaid
sequenceDiagram

    participant System as 系统

    Parent->>System: 验收通过

    System->>System: 计算积分
    Note over System: 积分 = 基础分 × 难度系数 × 连续加成
    
    System->>Child: 积分到账动画+余额更新
    System->>Parent: 通知：已验收

    Note over Child,Parent: --- 兑换环节 ---

    Child->>System: 查看积分商店
    System->>Child: 可用商品列表
    Child->>System: 点击「兑换」
    System->>System: 检查积分是否足够
    alt 积分不足
        System-->>Child: 「还差20分，继续加油！」
    else 积分足够
        System->>System: 扣减积分
        System->>Parent: 通知：孩子申请兑换「XX」
        Parent->>System: 确认/拒绝
        alt 确认
            System->>Child: 兑换成功
            Parent->>Child: 兑现奖励
        else 拒绝
            System->>System: 退还积分
            System-->>Child: 「家长暂时不同意，可以沟通一下」
        end
    end
```
