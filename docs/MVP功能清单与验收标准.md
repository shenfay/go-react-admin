## MVP 总验收场景（通关测试）

按以下脚本完整走一遍，每一步通过才算MVP达标：

```
1. 家长注册 → 创建孩子「小明」
2. 家长新建目标「掌握分数加减法」→ 关联归因卡(每周2次)
3. 孩子确认目标 → 目标激活
4. 第二天 → 系统自动分配今日卡片（含归因卡）
5. 孩子看到今日清单 → 伙伴显示当前形态+问候
6. 孩子完成归因卡 → 提交 → 伙伴弹跳庆祝
7. 家长收到验收通知 → 查看 → 通过
8. 孩子端收到 → XP+积分到账 → 伙伴状态变开心
9. 连续几天重复 → 伙伴光晕变化、连续天数增加
10. 第4次归因卡完成 → 目标进度100% → 目标达成
11. 伙伴全屏庆祝 → 积分到账 → 目标徽章颁发
12. 孩子用积分到商店兑换一个商品 → 家长确认 → 兑换成功
```
